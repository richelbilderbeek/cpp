# ([C++](Cpp.md)) [helgrind](CppHelgrind.md)

[helgrind](CppHelgrind.md) is a [tool](CppTool.md), part of [valgrind](CppValgrind.md), 
that is 'a thread debugger which finds data races in multithreaded
programs' [1].
